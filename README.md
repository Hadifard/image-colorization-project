# Modern Image Colorization (Deep Learning)

A modern image and real-time webcam colorization project using
**PyTorch** and **HuggingFace Transformers**.

## Features
- Automatic color detection
- High-quality colorization
- Real-time webcam support
- Pretrained Transformer model
- Fully open-source

## Model
- vit-based-colorization (HuggingFace)
- Transformer-based architecture

## Installation
pip install -r requirements.txt
